# Общее описание решения

Данный проект представляет собой файлы на языке `Python` расчитывающие периметр и площадь простых геометрических фигур.

Основные функции:
- Нахождение площадей для круга, прямоугольника, квадрата, треугольника 
- Нахождение периметров для круга, прямоугольника, квадрата, треугольника 


# Описание функций

## Функция 1 из файла circle.py: area
- Описание: Принимает радиус r, возвращает площадь круга с радиусом r.

## Функция 2 из файла circle.py: perimeter
- Описание: Принимает радиус r, возвращает периметр круга с радиусом r.

## Функция 1 из файла rectangle.py: area
- Описание: Принимает стороны a и b, возвращает площадь прямоугольника со сторонами a и b.

## Функция 2 из файла rectangle.py: perimeter
- Описание: Принимает стороны a и b, возвращает периметр прямоугольника со сторонами a и b.

## Функция 1 из файла square.py: area
- Описание: Принимает сторону a, возвращает площадь квадрата со стороной a.

## Функция 2 из файла square.py: perimeter
- Описание: Принимает сторону a, возвращает периметр квадрата со стороной a.

## Функция 1 из файла triangle.py: area
- Описание: Принимает сторону a и высоту h, возвращает площадь треугольника со стороной a и высотой h.

## Функция 2 из файла triangle.py: perimeter
- Описание: Принимает сторонами a,b,c и возвращает периметр треугольника с данными сторонами.
